## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/netflix-clone.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd netflix-clone
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm start
   ```
